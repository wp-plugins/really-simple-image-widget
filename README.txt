Easiest way to add image in your sidebar

== Description ==

Really Simple Image Widget is the easiest way to add image in your sidebar. Chose image from your Media, save it and done.


== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'really-simple-image-widget.zip' from your computer
1. Install
1. Activate the plugin on the WordPress Plugin Dashboard
1. Place the 'Really Simple Image Widget' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.

= Using FTP =

1. Extract 'really-simple-image-widget.zip' to your computer
1. Upload the 'really-simple-image-widget' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. Place the 'Really Simple Image Widget' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.
